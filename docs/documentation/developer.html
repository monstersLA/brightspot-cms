Model. Create your content interactively.

Brightspot CMS is built using the Dari, a powerful object persistence library that makes it easy to build complex content types and persist them to one or more database backends.

Use Dari to model content.

Rapidly build complex data models using Dari. Using simple annotations and standard Java types, anything can be modeled.

Dari supports inheritance at the database level.

A user interface for publishing data to your models is built automatically. New fields instantly show up on the CMS UI ready for data to be input.

{% highlight java %} @Renderer.Engine("JSP") @Renderer.Script("/WEB-INF/model/project.jsp") public class Project extends Content { @Required @ToolUi.Note("Examples: Perfect Sense") private String title; @ToolUi.Note("Examples: http://www.perfectsensedigital.com") private String url; @Indexed private List categories; private Client client; } {% endhighlight %}

Intuitive query API.

Dari provides a simple, yet powerful, API to query your object models.

Everything you expect is there: subqueries, count, order by, group by, limit.

{% highlight java %} // Query for all projects. List projects = Query.from(Project.class). select(); // Query for first 10 projects. PaginatedResult projects = Query.from(Project.class). select(0, 10); // Query for first 10 projects, starting with the Letter M. PaginatedResult projects = Query.from(Project.class). where("title startsWith 'M'"). select(0, 10); {% endhighlight %}

Structured document storage.

Dari provides a powerful structured document storage built on top of MySQL and Solr. You never need to update your database schema when using Dari.

Dari also saves all your data to Solr transparently giving you the ability to build robust search capabilities into your site.

{% highlight java %} Project project = new Project(); project.setTitle("Brightspot CMS Site"); project.setUrl("http://www.brightspotcms.com"); project.setClient(perfectsense); project.save(); {% endhighlight %}

Debug. Powerful performance debugging tools.

Dari provides powerful tools to debug performance issues directly on a page. See exactly which JSPs or Dari queries are slowing your site down. Quickly click to see the offending code or query right in your browser.

Get a performance overview.

Each page is broken down by events giving you a quick overview of how many queries are executed during the rendering of a page. This will help you quickly see if there are any performance issues on the page.

See problem areas.

Performance bubbles indicate how long each area of the page took to render. Red bubbles are a sure sign that a JSP or query needs to be reviewed.

Clicking a bubble takes you right to the associated JSP or query.

Get detailed information.

The page rendering events are displayed as a waterfall timeline allowing you to quickly drill down to see problem queries.

From the timeline you can click to see offending code, run an explain plan on MySQL or just execute the Dari query to see the results.
